# ☕ Coffee Corner - Multilingual Café Website

**Live Demo:** 🌐 [Coffee Corner Live](https://coffeeshop.gt.tc/)

---

## 📖 Project Overview

**Coffee Corner** is a **multilingual café website** built on **WordPress**, designed to improve the digital experience for cafés by providing a complete online menu and ordering system.  

Customers can:
- Browse categorized menu items (Drinks, Desserts, Breakfast)  
- Filter and sort products (price ↑↓, newest)  
- Add items to cart and complete checkout easily  

The platform also includes:
- **AI Chat Support** for instant recommendations and assistance  
- **User accounts** with saved preferences  
- **Analytics** for admin monitoring  
- **Responsive modern design** (mobile-first)

## 🛠️ Technologies & Tools

- **Platform:** WordPress  
- **Theme:** The Coffee Shop Theme + Elementor  
- **Plugins:** WooCommerce, Payment Gateway Plugin, AI Chat Plugin, TranslatePress  
- **Backend:** PHP (WordPress Core), MySQL Database  
- **Frontend:** HTML, CSS, JavaScript  
- **Development:** XAMPP Localhost  
- **Deployment:** InfinityFree Hosting  

---

## 🎯 Key Features

### Customer Features
- Browse and filter menu items  
- Add-to-cart & checkout system  
- Multilingual (English/Arabic)  
- AI chat assistant for support  
- User registration & login with saved preferences  

### Admin Features
- Manage products, categories, and orders  
- Access analytics and customer data  
- Manage plugins (WooCommerce, TranslatePress, AI Chat, Payment Gateway)  

### Additional Features
- Shipping rules: Free >100 EGP, 30 EGP otherwise  
- Fast loading and optimized performance  
- SEO-ready and secure transactions  
- Custom header & footer using Elementor  
- Clean modern café-themed design

## 👥 User Roles
1. **Customers:** Browse menu, add to cart, checkout, chat with AI  
2. **Registered Users:** Save favorites, manage accounts, faster checkout  
3. **Admins/Owners:** Manage menu, orders, users, and analytics  
4. **Staff:** Track and update orders  
5. **Developer:** Maintain and enhance the website  

---

## 🏆 Goals & Mission
Many cafés lack a digital system for menu browsing and online ordering. Coffee Corner provides:
- Easy online menu navigation  
- Product sorting & filtering options  
- User accounts with saved preferences  
- Simple checkout with shipping rules  
- AI-powered assistance to improve customer experience and boost online sales
